python包含六种序列类：list、tuple、string、unicode、buffer、xrange。其中xrange比较特殊，它是一个生成器，其他几个类型具有的一些序列特性对它并不合适。

一般具有序列结构的数据类型都可以使用index、len、max、min、in、+、\* 切片：
```python
>>> a = 'ILovePython'
>>> len(a)
11
>>> max(a)
'y'
>>> min(a)
'I'
>>> bool('o' in a)
True
>>> a + a
'ILovePythonILovePython'
>>> a * 3
'ILovePythonILovePythonILovePython'
>>> a[1:5]
'Love'
>>> a.index('y')
6
>>> a[5]
'P'
```
