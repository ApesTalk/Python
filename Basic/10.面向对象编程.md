

## 一、类和实例

- class小写，类名首字母大写，object表示继承的父类。
- 函数的``__init()__``方法第一个参数永远是self，表示创建的实例本身，
因此，在``__init__``方法内部，就可以把各种属性绑定到self，因为self就指向创建的实例本身。
- 在类中定义的函数只有一点不同，就是第一个参数永远是实例变量self，并且，调用时，不用传递该参数。
- Python允许对实例变量绑定任何数据。

```python
>>> class Student(object):
...     def __init__(self, name, score):
...             self.name = name
...             self.score = score
...     def print_score(self):
...             print('%s: %s' % (self.name, self.score))
...     def get_grade(self):
...             if self.score >= 90:
...                     return 'A'
...             elif self.score >= 80:
...                     return 'B'
...             else:
...                     return 'C'
... 
>>> bart = Student('Bart', 99)
>>> lambert = Student('Lambert', 88)
>>> bart
<__main__.Student object at 0x10b44e5c0>
>>> lambert
<__main__.Student object at 0x10b44e5f8>
>>> bart.print_score()
Bart: 99
>>> lambert.getScore()
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
AttributeError: 'Student' object has no attribute 'getScore'
>>> lambert.get_grade()
'B'
>>> lambert.age = 20
>>> lambert.age
20
>>> bart.age
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
AttributeError: 'Student' object has no attribute 'age'
```

## 二、访问限制

```python
>>> class Student(object):
...     def __init__(self, name, score):
...             self.__name = name
...             self.__score = score
...     def print_score(self):
...             print('%s: %s' % (self.__name, self.__score))
...     def get_name(self):
...             return self.__name
...     def get_score(self):
...             return self.__score
...     def set_score(self, score):
...             if 0 <= score <= 100:
...                     self.__score = score
...             else:
...                     raise ValueError('bad score')
... 
>>> bart = Student('Bart', 99)
>>> bart.name
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
AttributeError: 'Student' object has no attribute 'name'
>>> bart.get_name()
'Bart'
>>> bart.get_score()
99
>>> bart.set_score(100)
>>> bart.get_score()
100
>>> bart._Student__name
'Bart'
>>> bart.__name = 'Test'
>>> bart.__name
'Test'
>>> bart._Student__name
'Bart'
```

在变量前面加上``__``表示私有变量，外部无法访问。name属性在编译器内部已改为``_Studen__name``属性，
``bart.__name = 'Test'``结果是给bart新增了变量``__name``。


## 三、继承和多态

```python
>>> class Animal(object):
...     def run(self):
...             print('Animal is running...')
... 
>>> class Dog(Animal):
...     def run(self):
...             print('Dog is running...')
... 
>>> class Cat(Animal):
...     def run(self):
...             print('Cat is running...')
... 
>>> def run_twice(animal):
...     animal.run()
...     animal.run()
... 
>>> run_twice(Dog())
Dog is running...
Dog is running...
>>> run_twice(Cat())
Cat is running...
Cat is running...
>>> class Tortoise(Animal):
...     def run(self):
...             print('Tortoise is running...')
... 
>>> run_twice(Tortoise())
Tortoise is running...
Tortoise is running...
>>> class Timer(object):
...     def run(self):
...             print('Start...')
... 
>>> # 只需保证传入的对象有run方法即可
>>> run_twice(Timer())
Start...
Start...
```

**开闭原则**：对扩展开放（可创建新的Animal子类），对修改封闭（创建子类后不需要修改Animal的run方法即可调用）。

**鸭子类型**：动态语言的“鸭子类型”，它并不要求严格的继承体系，一个对象只要“看起来像鸭子，走起路来像鸭子”，那它就可以被看做是鸭子。

## 五、实例属性和类属性

Python可以通过实例变量或者self给实例绑定任意属性。直接在class中定义的属性属于类的属性。

```python
>>> class Student(object):
...     name = 'Student' # 类属性
... 
>>> s = Student()
>>> print(s.name) # 因为实例没有name属性，会继续查找class的name属性
Student
>>> s.name = 'Lambert' # 给实例绑定name属性
>>> s.name
'Lambert'
>>> Student.name
'Student'
>>> del s.name # 删除实例name属性
>>> s.name
'Student'
>>> del Student.name # 删除类的name属性
>>> Student.name
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
AttributeError: type object 'Student' has no attribute 'name'
```

## 六、使用``__slots__``限制实例可绑定的属性

给对象或类动态绑定方法：

```python
>>> class Student(object):
...     pass
... 
>>> s = Student()
>>> s.name = 'Lambert' # 动态给实例绑定属性
>>> s.name
'Lambert'
>>> def set_age(self, age):
...     self.age = age
... 
>>> from types import MethodType
>>> s.set_age = MethodType(set_age, s) # 给实例绑定方法
>>> s.set_age(20) # 调用实例方法
>>> s.age
20
>>> s2 = Student() # 新实例对象
>>> s2.set_age(22) # 尝试调用新对象的方法
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
AttributeError: 'Student' object has no attribute 'set_age'
>>> def set_score(self, score):
...     self.score = score
... 
>>> Student.set_score = set_score # 给class绑定方法
>>> s.set_score(100)
>>> s.score
100
>>> s2.set_score(99)
>>> s2.score
99
```

``__slots__``可以限制给类动态添加的属性名称，只对当前类有效，对子类无效。
如果子类中也定义的有``__slots__``，那么子类可绑定的属性是子类的``__slots__``加上父类的``__slots__``。

```python
>>> class Student(object):
...     __slots__ = ('name', 'age') # 用tuple定义允许绑定的属性名称
... 
>>> s = Student()
>>> s.name = 'Lamber'
>>> s.age = 20
>>> s.score = 100 # 尝试绑定不再__slots__中声明的属性
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
AttributeError: 'Student' object has no attribute 'score'
>>> class GraduateStudent(Student):
...     pass
... 
>>> g = GraduateStudent()
>>> g.score = 999 # 尝试给未声明__slots__的子类绑定属性
>>> g.score
999
>>> class GraduateStudent(Student):
...     __slots__ = ('score') # 子类中新增一个属性名称声明
... 
>>> g = GraduateStudent()
>>> g.score = 1000
>>> g.name = 'Graduate'
>>> g.name
'Graduate'
>>> g.gender = 'male' # 尝试绑定一个新属性
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
AttributeError: 'GraduateStudent' object has no attribute 'gender'
```

## 七、使用@property

@property装饰器负责将方法变成属性调用

```python
>>> class Student(object):
...     @property
...     def score(self):
...             return self._score
...     @score.setter
...     def score(self, value):
...             if not isinstance(value, int):
...                     raise ValueError('score must be an integer')
...             if value < 0 or value > 100:
...                     raise ValueError('score must between 0 ~ 100')
...             self._score = value
... 
>>> s = Student()
>>> s.score = 60
>>> s.score = 999
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
  File "<stdin>", line 10, in score
ValueError: score must between 0 ~ 100
```

只定义getter没有setter表示只读属性：

```python
>>> class Student(object):
...     @property
...     def birth(self):
...             return self._birth
...     @birth.setter
...     def birth(self, value):
...             self._birth = value
...     @property
...     def age(self):
...             return 2017 - self._birth
... 
>>> s = Student()
>>> s.birth = 1990
>>> s.birth
1990
>>> s.age
27
```

## 八、多重继承

Python允许多继承，如果父类中包含重复的方法，那么会继承距离最近的父类的方法。

```python
>>> class Father():
...     def func(self):
...             print('生父打儿子')
... 
>>> class Laowang():
...     def func(self):
...             print('老王打儿子')
... 
>>> class StepFather():
...     def func(self):
...             print('继父打儿子')
... 
>>> class Son(Father, Laowang, StepFather):
...     pass
... 
>>> s = Son()
>>> s.func()
生父打儿子
>>> class Son(Laowang, Father, StepFather):
...     pass
... 
>>> s = Son()
>>> s.func()
老王打儿子
```
