## 一、[Python函数库](https://docs.python.org/3/library/functions.html) 

```
>>> abs(-1024)
1024
>>> max(1,20,0,-200)
20
>>> min(20,-2,100)
-2
>>> int('1024')
1024
>>> float('10.24')
10.24
>>> str(1024)
'1024'
>>> hex(1024)
'0x400'
>>> bool(1)
True
>>> bool('')
False
# 变量a指向abs函数
>>> a = abs
>>> a(-256)
256
```

## 二、自定义函数

```
>>> def my_abs(x):
...     if x >= 0:
...             return x
...     else:
...             return -x
... 
>>> # 这里连续输入两次回车
>>> my_abs(-1024)
1024
>>> my_abs(1024)
1024
```
